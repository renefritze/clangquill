# C++ API (dogfood)

clangquill documents its own C++ — these pages are generated by running the
clangquill Sphinx extension over the project's public IR data-model headers
(`src/cpp/model/*.hpp`) at docs-build time. They are a live demonstration of the
tool producing C++-domain API docs for a real codebase.

The configuration that drives this lives in `docs/conf.py` (`clangquill_input`,
`clangquill_include_dirs`, `clangquill_output_dir`, `clangquill_std`).
